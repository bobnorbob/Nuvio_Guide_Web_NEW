Register for Accounts and Obtain API Keys +
Before configuring Nuvio, AIOStreams, and AIOMetadata, register for the required services and obtain your API keys.

Tip: Create a text document to store your API keys for easy access.

Nuvio +
Your media server dashboard for streaming and managing content.
Real-Debrid (Paid Account Required) +
A "debrid" service that unlocks high-speed downloads and streaming from file-hosting services.
Trakt (Free Account) +
Tracks your viewing progress across devices.
No API key is required, but you will connect your account in later steps.
Note: You can also find great curated lists that are kept up to date (e.g. New Releases, Trending, etc.). Have a look around the site, add shows and movies to your Watchlist, and "like" some lists you find interesting so we can build a nice Home screen in Nuvio later.
TMDB (The Movie Database) +
Provides metadata for movies and TV shows, including posters, descriptions, and ratings.
TVDB +
Offers detailed TV show metadata, including episode guides and fanart.
Fanart.tv +
Provides high-quality fanart and posters for movies and TV shows.
RPDB (RatingPosterDB) +
Enhances your media library with user-rated posters and ratings.
